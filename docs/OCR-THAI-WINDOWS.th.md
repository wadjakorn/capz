# เปิดใช้การอ่านตัวอักษรภาษาไทย (OCR) บน Windows

> เอกสารนี้เขียนเป็นภาษาไทยแบบเข้าใจง่าย สำหรับผู้ใช้ทั่วไป (ไม่ใช่สายเทคนิค)

## ปัญหานี้คืออะไร?

capz ใช้ระบบอ่านตัวอักษรในรูปภาพ (OCR) ของ Windows เอง
แต่ Windows **บางเครื่องไม่ได้ติดตั้ง "ชุดภาษาไทย" มาให้ตั้งแต่แรก**

ถ้าเครื่องของคุณยังไม่มีชุดภาษาไทย เมื่อกดปุ่ม **"Detect text"** (อ่านตัวอักษร) ในรูปที่มีภาษาไทย
capz จะขึ้นข้อความเตือนว่า:

> **Thai text recognition isn't available on this system — install the Thai language pack in Windows Settings.**
> *(แปลว่า: เครื่องนี้ยังอ่านภาษาไทยไม่ได้ — กรุณาติดตั้งชุดภาษาไทยในการตั้งค่า Windows)*

ภาษาอังกฤษยังอ่านได้ปกติ แค่ภาษาไทยยังไม่ได้ ทำตามขั้นตอนด้านล่างเพื่อเปิดใช้งาน

---

## วิธีติดตั้งชุดภาษาไทย (ทำครั้งเดียว)

> ⏱️ ใช้เวลาประมาณ 2–5 นาที และต้องต่ออินเทอร์เน็ต

1. กดปุ่ม **Start** (โลโก้ Windows มุมซ้ายล่าง) → เลือก **Settings** (รูปเฟือง ⚙️)
2. ไปที่ **Time & language** (เวลาและภาษา)
3. เลือก **Language & region** (ภาษาและภูมิภาค)
4. ที่หัวข้อ **Preferred languages** กดปุ่ม **Add a language** (เพิ่มภาษา)
5. พิมพ์หรือเลือก **"ไทย" / "Thai"** → กด **Next** (ถัดไป)
6. หน้าถัดมาจะมีรายการให้ติ๊กเลือก — **สำคัญ: ติ๊กให้ครบ**
   - ✅ **Language pack** (ชุดภาษา)
   - ✅ **Optical character recognition** (การรู้จำตัวอักษร / OCR) ← อันนี้คือตัวที่ทำให้ capz อ่านไทยได้
   - (ตัวเลือกอื่น เช่น Text-to-speech ไม่จำเป็น จะติ๊กหรือไม่ก็ได้)
7. กด **Install** (ติดตั้ง) แล้วรอจนเสร็จ (แถบสถานะวิ่งจนครบ)
8. **ปิดแล้วเปิด capz ใหม่** (คลิกขวาไอคอน capz มุมขวาล่าง → Quit แล้วเปิดโปรแกรมอีกครั้ง)

เสร็จแล้ว! ลองกด **"Detect text"** บนรูปภาษาไทยอีกครั้ง คราวนี้ควรอ่านได้แล้ว

> 💡 ไม่จำเป็นต้องเปลี่ยนภาษาหน้าจอ Windows เป็นไทย — แค่ "เพิ่มภาษา" พร้อมตัวเลือก OCR ก็พอ

---

## ยังใช้ไม่ได้? ตรวจสอบดังนี้

- **ติ๊ก "Optical character recognition" ครบหรือยัง?**
  กลับไปที่ Language & region → คลิกที่ **ไทย** → **Language options** (ตัวเลือกภาษา)
  แล้วดูว่าหัวข้อ OCR ขึ้นว่า *Installed* (ติดตั้งแล้ว) หรือไม่ ถ้ายังไม่มี ให้กดติดตั้ง
- **เปิด capz ใหม่แล้วหรือยัง?** ต้องปิด–เปิดโปรแกรมหลังติดตั้งภาษาเสร็จ
- **ต่ออินเทอร์เน็ตขณะติดตั้งหรือไม่?** ชุดภาษาต้องดาวน์โหลดจากอินเทอร์เน็ต
